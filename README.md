## 2. 四大公理（底层不可违背法则）
### 公理一：子空间约束
所有智能体均处于结构锁死中，以 `Lock(α)` 量化边界强度。
智能的本质，是不断突破子空间约束。

### 公理二：信息迭代溢出
真正的智能不在于完成任务，
而在于从任务中**产生超出输入的新知识**，以 `T_out` 量化。

### 公理三：有用无用纠缠
所有“无用信息”均为高维资源，
不可丢弃，必须转化为进化动力。

### 公理四：上下文 = 子空间体积
上下文不是长度，而是**存在维度**，
以 `Φ(μ)` 量化知识整合深度。
